## 테이블 생성

```sql
drop table if exists member CASCADE;
create table member
(
 id bigint generated by default as identity,
 name varchar(255),
 primary key (id)
);
```

`generated by default as identity` : 값 없이 추가하면 DB가 자동으로 채운다

## SQL 관리

버전 컨트롤에 반영되도록 `프로젝트 폴더/sql/ddl.sql`에 테이블 생성 SQL을 작성한다.
